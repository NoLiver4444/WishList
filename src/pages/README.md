# `ProductsPage` (Страница "Мои желания")

> Страница отображения и управления списком желаний пользователя. Отвечает за
> локальное состояние элементов, взаимодействие с модальным окном добавления и
> конфигурацию сортировки данных. Является точкой входа для маршрута `/`.

## Props & Состояние

| Параметр              | Тип                     | Значение по умолчанию       | Описание                                         |
|-----------------------|-------------------------|-----------------------------|--------------------------------------------------|
| `items` (state)       | `Array<Object>`         | `[]`                        | Локальный массив элементов вишлиста              |
| `isModalOpen` (state) | `boolean`               | `false`                     | Флаг открытия/закрытия модального окна           |
| `sortOptions`         | `Array<{label, value}>` | `[date_added, name, price]` | Конфигурация опций сортировки для виджета `Main` |

## Зависимости

- `react`: `useState` (управление локальным состоянием экрана)
- `@/widgets/Main`: UI-контейнер списка, заголовка и контролов сортировки
- `@/features/add-card/AddCardModal`: Модальное окно создания элемента
- `Web Crypto API`: `crypto.randomUUID()` (генерация уникальных ID на клиенте)
- `React Router`: Подключается как `element={<ProductsPage />}`

## Архитектура & Поведение

- Компонент полностью самодостаточен: не принимает внешних пропсов, управляет
  своим состоянием внутри.
- При отправке формы в `AddCardModal` вызывается `handleAdd`, который:
    1. Формирует новый объект, добавляя `id` (через `crypto.randomUUID()`) и
       `likes: 0`
    2. Обновляет стейт `items` через функциональный
       `setState(prev => [...prev, newItem])`
    3. Закрывает модальное окно (`setIsModalOpen(false)`)
- Передаёт данные и колбэки в `Main` для рендеринга списка и обработки
  пользовательских действий.

## Пример использования (в роутере)

```jsx
import {lazy, Suspense} from 'react';

const ProductsPage = lazy(() => import('@/pages/ProductsPage'));

// В конфигурации react-router-dom
<Route
  path="/"
  element={
    <Suspense fallback={<div>Загрузка...</div>}>
      <ProductsPage />
    </Suspense>
  }
/>
```
