# `CardList` (Компонент списка карточек)

Рендерит список карточек с кнопкой добавления нового элемента вверху.
Динамически подставляет заголовок кнопки в зависимости от переданного типа.
Используется для отображения массивов данных в унифицированном виде.

## Props

| Prop         | Тип                    | Default | Описание                                                             |
|--------------|------------------------|---------|----------------------------------------------------------------------|
| `type`       | `string`               | `-`     | Тип сущности. Определяет текст кнопки добавления и передаётся в Card |
| `items`      | `Array<Object>`        | `[]`    | Массив данных для рендеринга карточек                                |
| `onAddClick` | `() => void`           | `-`     | Обработчик клика по кнопке добавления                                |
| `onLike`     | `(id: string) => void` | `-`     | Обработчик лайка (объявлен в сигнатуре, но не привязан к UI)         |

## Зависимости

- `@/features/add-card/ui/AddCard`: Компонент кнопки добавления нового элемента
- `@/shared/ui/Card`: Универсальный компонент отображения отдельной карточки

## Архитектура и поведение

- Компонент принимает массив `items` и итерирует его через `.map()`, передавая
  поля `id`, `name`, `date`, `counts`, `wishes` и `type` в `<Card />`.
- Заголовок кнопки добавления определяется через объект `titles`. Если
  переданный `type` отсутствует в мапе, используется значение по умолчанию
  `'Добавить'`.
- Присутствует условный рендеринг для пустого состояния, который в текущей
  реализации возвращает пустой фрагмент.
- Используется значение по умолчанию `items = []`, что предотвращает ошибки при
  попытке вызвать `.map()` на `undefined`.

## Пример использования

```jsx
<CardList
  type="wishes"
  items={[{id: '1', name: 'Книга', date: '2024-05-10', counts: 2, wishes: 15}]}
  onAddClick={() => openModal()}
  onLike={(id) => handleLike(id)}
/>
```
