# `Footer` (Подвал приложения)

Компонент нижней панели сайта. Содержит навигацию на главную страницу,
информацию о бренде, копирайт и внешние ссылки на поддержку и репозиторий
проекта.

## Props

Компонент не принимает внешних пропсов. Является статическим виджетом.

## Зависимости

- `react-router-dom`: `Link` (клиентская навигация без перезагрузки)
- `./Footer.module.css`: CSS-модули для стилизации

## Архитектура и поведение

- Использует семантический тег `<footer>` для корректной разметки и доступности.
- Внутренняя ссылка на главную страницу реализована через `<Link to="/">` для
  работы SPA-роутинга.
- Содержит внешние ссылки с `target="_blank"` для перехода в Telegram (
  поддержка) и GitHub (о проекте).
- Включает статические данные: название проекта "Wish-piece" и копирайт "© 2026
  Los huilos".
- Не управляет состоянием, предназначен для чистого отображения информации и
  навигации.

## Пример использования

```jsx
import Footer from '@/widgets/Footer';

const App = () => (
  <div className="app">
    <header>...</header>
    <main>...</main>
    <Footer />
  </div>
);
```
