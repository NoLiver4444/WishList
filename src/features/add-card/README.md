# `AddCardModal` (Модальное окно создания сущности)

Управляемое модальное окно с динамической формой. Генерирует поля ввода на
основе конфигурации, выполняет валидацию по правилам из конфига и закрывается по
клику вне области, нажатию Escape или Enter.

## Props

| Prop       | Тип                                    | Default | Описание                                   |
|------------|----------------------------------------|---------|--------------------------------------------|
| `isOpen`   | `boolean`                              | `false` | Флаг видимости модалки                     |
| `onClose`  | `() => void`                           | `-`     | Обработчик закрытия окна                   |
| `onSubmit` | `(formData: Object) => void`           | `-`     | Колбэк успешной отправки формы             |
| `type`     | `'wishes' \| 'wishlists' \| 'friends'` | `-`     | Тип сущности. Определяет набор полей формы |
| `title`    | `string`                               | `-`     | Заголовок модального окна                  |

## Зависимости

- `react`: `useCallback`, `useMemo`, `useRef`, `useState`
- `./AddCardModal.config.js`: Конфигурация полей для разных типов сущностей
- `@/shared/hooks/useClickOutside`: Закрытие при клике вне модалки
- `@/shared/hooks/useEscClose`: Закрытие по клавише Escape
- `@/shared/hooks/useEnterPress`: Отправка формы по клавише Enter
- `./AddCardModal.module.css`: CSS-модули стилизации

## Архитектура и поведение

- Конфигурация полей загружается через `useMemo` на основе пропса `type`. При
  отсутствии конфига для типа используется fallback на `wishes`.
- Начальное состояние формы инициализируется пустыми строками на основе ключей
  из конфигурации.
- Валидация проверяет обязательные поля и соответствие регулярным выражениям.
  Ошибки отображаются под каждым полем.
- При изменении значения поля ошибка автоматически сбрасывается для этого поля.
- Закрытие окна полностью очищает состояние формы и ошибок перед вызовом
  внешнего `onClose`.
- Рендерится условно: если `isOpen === false`, возвращает `null`.
- Поддерживает три типа полей: стандартный `input`, `select` (для приватности) и
  `textarea`.

## Пример использования

```jsx
<AddCardModal
  isOpen={isModalOpen}
  onClose={() => setIsModalOpen(false)}
  onSubmit={(formData) => handleCreateItem(formData)}
  type="wishes"
  title="Добавить новое желание"
/>
```
