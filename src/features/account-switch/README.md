# `AccountSwitcher` (Переключатель аккаунтов)

Компонент управления активным пользователем. Содержит мок-данные аккаунтов,
отслеживает текущий выбор через локальное состояние и делегирует рендеринг
интерфейса выбора компоненту `Menu`.

## Props

Компонент не принимает внешних пропсов. Работает с внутренним состоянием и
статическими данными.

## Зависимости

- `react`: `useState`
- `@/shared/ui/Menu`: Компонент меню выбора аккаунта
- `@/shared/assets/govishka.png`: Изображение аватара

## Архитектура и поведение

- Хранит массив пользователей `USERS` и флаг уведомлений `HAS_UNREAD` локально.
- Инициализирует состояние `currentUser` первым элементом массива.
- При выборе пользователя через `Menu` обновляет состояние через
  `setCurrentUser`.
- Передает в `Menu` текущий аккаунт, полный список пользователей, обработчик
  выбора и статус непрочитанных уведомлений.

## Примеры использования

```jsx
import AccountSwitcher from '@/features/account-switch/AccountSwitcher';

const Header = () => (
  <header>
    <AccountSwitcher />
  </header>
);
```
