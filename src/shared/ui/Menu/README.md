# `Menu` (Главное меню шапки)

Компонент управления выпадающими списками аватара и уведомлений, а также
модальным окном профиля. Отвечает за переключение состояний, обработку кликов
вне элементов и анимацию появления/скрытия контента.

## Props

| Prop           | Тип                      | Default | Описание                               |
|----------------|--------------------------|---------|----------------------------------------|
| `currentUser`  | `Object`                 | `-`     | Данные текущего пользователя           |
| `users`        | `Array<Object>`          | `-`     | Список доступных аккаунтов             |
| `onSelectUser` | `(user: Object) => void` | `-`     | Обработчик выбора аккаунта             |
| `hasUnread`    | `boolean`                | `false` | Флаг наличия непрочитанных уведомлений |

## Зависимости

- `react`: `useRef`, `useState`
- `framer-motion`: `AnimatePresence`
- `lucide-react`: `Bell`, `ChevronDown`
- `@/shared/hooks/useClickOutside`
- `@/shared/ui/Dropdowns`, `@/shared/ui/ProfilePopup`
- `./Menu.module.css`

## Архитектура и поведение

- Управляет двумя независимыми дропдаунами (`avatar`, `notification`) и
  состоянием полноэкранного попапа профиля.
- При клике на активный элемент дропдаун закрывается, иначе открывается.
- Использует `useClickOutside` для автоматического закрытия открытых меню при
  клике вне их области.
- Рендерит компоненты через `AnimatePresence` для плавной анимации
  монтирования/размонтирования.
- При открытии полного профиля скрывает текущий дропдаун и меняет состояние
  `isProfileOpen`.

## Пример использования

```jsx
import Menu from '@/shared/ui/Menu';

<Menu
  currentUser={activeUser}
  users={allUsers}
  onSelectUser={handleSwitch}
  hasUnread={true}
/>
```
