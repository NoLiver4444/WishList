# `AddCard` (UI-компонент кнопки добавления)

Отображает карточку-кнопку с иконкой плюса и текстовым заголовком. Используется
в списках как точка входа для вызова модального окна создания сущности.

## Props

| Prop      | Тип          | Default | Описание                                                |
|-----------|--------------|---------|---------------------------------------------------------|
| `title`   | `string`     | `-`     | Текст кнопки                                            |
| `onClick` | `() => void` | `-`     | Обработчик клика по кнопке                              |
| `type`    | `string`     | `-`     | Тип сущности. Используется для динамического CSS-класса |

## Зависимости

- `lucide-react`: `Plus` (иконка добавления)
- `./AddCard.module.css`: CSS-модули стилизации

## Архитектура и поведение

- Рендерит элемент списка `<li>` с кнопкой внутри.
- Применяет комбинированный CSS-класс на основе пропса `type` для кастомизации
  стилей под разные сущности.
- Кнопка имеет явный `type="button"` и `tabIndex={0}` для корректной работы с
  клавиатурой и предотвращения отправки форм.
- Иконка `Plus` имеет фиксированный размер и обводку для единообразия в списке.

## Пример использования

```jsx
<AddCard
  title="Добавить желание"
  onClick={() => setModalOpen(true)}
  type="wishes"
/>
```
