# `Card` (Компонент карточки)

Универсальная карточка для отображения элементов вишлиста, желаний или друзей.
Поддерживает условный рендеринг даты, счётчиков и кастомного контента через
`children`.

## Props

| Prop          | Тип          | Default | Описание                                                                           |
|---------------|--------------|---------|------------------------------------------------------------------------------------|
| `name`        | `string`     | `-`     | Заголовок карточки                                                                 |
| `date`        | `string`     | `-`     | Дата (отображается для `wishes` и `wishlists`)                                     |
| `counts`      | `number`     | `-`     | Количество подарков (отображается для `wishlists`)                                 |
| `children`    | `ReactNode`  | `-`     | Кастомный контент внутри карточки                                                  |
| `onMenuClick` | `() => void` | `-`     | Обработчик клика по кнопке меню                                                    |
| `type`        | `string`     | `-`     | Тип сущности (`wishes`, `wishlists`, `friends`). Влияет на стили и видимость полей |

## Зависимости

- `lucide-react`: `Ellipsis` (иконка меню)
- `./Card.module.css`: CSS-модули

## Архитектура и поведение

- Рендерится как элемент списка `<li>`.
- Условно отображает дату для типов `wishes` и `wishlists`.
- Условно отображает блок с количеством подарков только для `wishlists`.
- Кнопка с иконкой `Ellipsis` вызывает `onMenuClick` для открытия контекстного
  меню.
- Поддерживает слот `children` для гибкого наполнения содержимого карточки.

## Пример использования

```jsx
<Card
  name="PlayStation 5"
  date="2026-12-01"
  type="wishes"
  onMenuClick={() => setMenuOpen(true)}
/>
```
