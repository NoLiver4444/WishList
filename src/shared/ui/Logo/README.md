# `Logo` (Логотип приложения)

Кликабельный логотип, ведущий на главную страницу. Используется в шапке и других
местах для брендинга и быстрой навигации.

## Props

Компонент не принимает внешних пропсов.

## Зависимости

- `react-router-dom`: `Link`
- `../../../../public/logo.svg`: SVG-изображение логотипа
- `./Logo.module.css`: CSS-модули

## Архитектура и поведение

- Обёрнут в `Link` с маршрутом `/` для SPA-навигации.
- Рендерит SVG-иконку через статический импорт из папки `public`.
- Атрибут `alt` содержит описание для доступности.
- Не управляет состоянием, является статическим презентационным компонентом.

## Пример использования

```jsx
import Logo from '@/shared/ui/Logo';

<header>
  <Logo />
</header>
```
