# `Navigation.config.js` (Конфигурация навигации)

Статический конфигурационный файл, определяющий структуру основного меню
приложения. Экспортирует массив объектов маршрутов для использования в
компоненте `Navigation`.

## Структура данных

| Поле    | Тип         | Описание                             |
|---------|-------------|--------------------------------------|
| `id`    | `string`    | Уникальный идентификатор пункта меню |
| `label` | `string`    | Отображаемый текст пункта            |
| `path`  | `string`    | Маршрут для `react-router-dom`       |
| `icon`  | `Component` | Иконка из `lucide-react`             |

## Зависимости

- `lucide-react`: `Calendar`, `ClipboardList`, `Heart`, `Users`

## Архитектура и поведение

- Содержит только данные, не управляет состоянием и не рендерит JSX.
- Используется для централизованного управления пунктами меню, что упрощает
  добавление новых разделов и синхронизацию с роутером.
- Позволяет `Navigation.jsx` динамически генерировать `HeaderButton` и
  отслеживать активный маршрут.

## Пример использования

```jsx
import {NAV_ITEMS} from './Navigation.config.js';

NAV_ITEMS.map(item => <HeaderButton
  key={item.id}
  item={item}
/>)
```

# `Navigation` (Панель навигации)

Компонент основного меню приложения с анимированным индикатором активного
раздела. Динамически вычисляет положение и ширину капсулы-подсветки на основе
текущего URL.

## Props

Компонент не принимает внешних пропсов.

## Зависимости

- `react`: `useEffect`, `useRef`, `useState`
- `react-router-dom`: `useLocation`
- `framer-motion`: `motion`
- `./Navigation.config.js`
- `@/shared/ui/HeaderButton`
- `./Navigation.module.css`

## Архитектура и поведение

- Отслеживает изменение маршрута через `useLocation`.
- При изменении пути или изменении размеров контейнера пересчитывает позицию и
  ширину активного элемента через `getBoundingClientRect`.
- Использует `ResizeObserver` для реактивного обновления положения капсулы при
  изменении макета.
- Анимация перемещения капсулы реализована через `motion.span` с параметрами
  пружинной физики.
- Рендерит список `HeaderButton` на основе `NAV_ITEMS`, сохраняя ссылки на
  DOM-элементы в `itemRefs`.

## Пример использования

```jsx
import Navigation from '@/shared/ui/Navigation';

<header>
  <Navigation />
</header>
```
