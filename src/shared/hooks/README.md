# `src/shared/hooks/` (Кастомные хуки)

Содержит переиспользуемые хуки для управления темой оформления, обработки
клавиатурных событий и отслеживания взаимодействий с DOM вне целевых элементов.

---

## `useTheme`

> Управление темой приложения с синхронизацией в localStorage и поддержкой
> системных настроек операционной системы.

### Возвращаемое значение

| Значение   | Тип                       | Описание                                                     |
|------------|---------------------------|--------------------------------------------------------------|
| `theme`    | `string`                  | Текущая тема (`'light'`, `'dark'`, `'contrast'`, `'system'`) |
| `setTheme` | `(value: string) => void` | Функция переключения и сохранения темы                       |

### Зависимости

- `react`: `useEffect`, `useState`
- `localStorage`: Сохранение пользовательского выбора
- `window.matchMedia`: Отслеживание системной цветовой схемы

### Архитектура и поведение

- Инициализирует стейт из `localStorage` или использует `'system'` по умолчанию.
- При каждом изменении темы применяет атрибут `data-theme` к
  `document.documentElement`.
- Если выбрана `'system'`, подписывается на изменения медиа-запроса
  `prefers-color-scheme` и автоматически обновляет атрибут темы.
- При подписке на медиа-запрос возвращает функцию очистки для удаления слушателя
  при изменении темы или размонтировании.

### Пример использования

```jsx
import {useTheme} from '@/shared/hooks/useTheme';

const ThemeSwitcher = () => {
  const {theme, setTheme} = useTheme();

  return (
    <button onClick={() => setTheme(theme === 'light' ? 'dark' : 'light')}>
      Переключить тему
    </button>
  );
};
```

## `useEnterPress`

> Хук для вызова обработчика при нажатии клавиши Enter. Слушатель активируется
> только при условии `isOpen === true`.

### Аргументы

| Аргумент       | Тип          | По умолчанию | Описание                              |
|----------------|--------------|--------------|---------------------------------------|
| `handleSubmit` | `() => void` | `-`          | Функция, вызываемая при нажатии Enter |
| `isOpen`       | `boolean`    | `true`       | Флаг активности слушателя клавиатуры  |

### Зависимости

- `react`: `useEffect`

### Архитектура и поведение

- Добавляет слушатель `keydown` на объект `document`.
- Проверяет условие `e.key === 'Enter'` и флаг `isOpen`.
- Вызывает `handleSubmit` при совпадении условий.
- Автоматически удаляет слушатель при размонтировании компонента или изменении
  зависимостей.

### Пример использования

```jsx
import {useEnterPress} from '@/shared/hooks/useEnterPress';

const Modal = ({isOpen, onClose, onSubmit}) => {
  useEnterPress(onSubmit, isOpen);

  return <div>Контент модального окна</div>;
};
```

## `useEscClose`

> Хук для вызова обработчика закрытия при нажатии клавиши Escape. Слушатель
> активируется только при условии `isOpen === true`.

### Аргументы

| Аргумент  | Тип          | По умолчанию | Описание                             |
|-----------|--------------|--------------|--------------------------------------|
| `onClose` | `() => void` | `-`          | Функция закрытия                     |
| `isOpen`  | `boolean`    | `true`       | Флаг активности слушателя клавиатуры |

### Зависимости

- `react`: `useEffect`

### Архитектура и поведение

- Добавляет слушатель `keydown` на объект `document`.
- Проверяет условие `e.key === 'Escape' || e.key === 'Esc'` и флаг `isOpen`.
- Вызывает `onClose` при совпадении условий.
- Автоматически удаляет слушатель при размонтировании компонента или изменении
  зависимостей.

### Пример использования

```jsx
import {useEscClose} from '@/shared/hooks/useEscClose';

const Dropdown = ({isOpen, onClose}) => {
  useEscClose(onClose, isOpen);

  return isOpen ? <div>Выпадающее меню</div> : null;
};
```

## `useClickOutside`

> Хук для отслеживания кликов или касаний вне указанного DOM-элемента или
> массива элементов.

### Аргументы

| Аргумент  | Тип                                         | Описание                                    |
|-----------|---------------------------------------------|---------------------------------------------|
| `refs`    | `React.RefObject \| React.RefObject[]`      | Ссылка или массив ссылок на DOM-элементы    |
| `handler` | `(event: MouseEvent \| TouchEvent) => void` | Функция, вызываемая при клике вне элементов |

### Зависимости

- `react`: `useEffect`

### Архитектура и поведение

- Нормализует входной параметр `refs` в массив для единообразной обработки
  одиночных и групповых ссылок.
- Подписывается на события `mousedown` и `touchstart` на `document`.
- Проверяет, находится ли `event.target` внутри любого из указанных элементов
  через метод `.contains()`.
- Если клик или касание произошли снаружи, вызывает `handler(event)`.
- Корректно очищает оба слушателя при размонтировании компонента или изменении
  ссылок/обработчика.

### Пример использования

```jsx
import {useRef} from 'react';
import {useClickOutside} from '@/shared/hooks/useClickOutside';

const Tooltip = ({onClose}) => {
  const tooltipRef = useRef(null);

  useClickOutside(tooltipRef, onClose);

  return <div ref={tooltipRef}>Всплывающая подсказка</div>;
};
```
