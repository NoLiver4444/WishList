# `SortCardsForm` (Форма сортировки)

Компонент выбора параметра сортировки. Отображает список доступных опций в виде
кнопок, подсвечивает активный вариант и вызывает обработчик при изменении
выбора.

## Props

| Prop           | Тип                       | Default | Описание                              |
|----------------|---------------------------|---------|---------------------------------------|
| `options`      | `Array<{value, label}>`   | `-`     | Массив доступных вариантов сортировки |
| `activeSort`   | `string`                  | `-`     | Значение текущей активной сортировки  |
| `onSortChange` | `(value: string) => void` | `-`     | Колбэк изменения выбранного параметра |

## Зависимости

- `./SortCardsForm.module.css`: CSS-модули

## Архитектура и поведение

- Итерирует массив `options` через `.map()` и рендерит каждый вариант как `<li>`
  с кнопкой внутри.
- Сравнивает `activeSort` с `option.value` для применения CSS-класса `isActive`
  к активной кнопке.
- При клике вызывает `onSortChange(option.value)`.
- Использует `type="button"` для предотвращения стандартного поведения формы.
- Не управляет внутренним состоянием, полностью контролируется родительским
  компонентом.

## Примеры использования

```jsx
<SortCardsForm
  options={[
    {label: 'По дате', value: 'date'},
    {label: 'По имени', value: 'name'}
  ]}
  activeSort="date"
  onSortChange={(val) => console.log('Выбрано:', val)}
/>
```
