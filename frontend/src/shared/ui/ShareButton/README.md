# `ShareButton` (Кнопка "Поделиться")

Презентационный компонент кнопки для функции шаринга. В текущей реализации не
содержит логики, предназначен для дальнейшей интеграции с Web Share API или
кастомным диалогом.

## Props

Компонент не принимает внешних пропсов.

## Зависимости

- `./ShareButton.module.css`

## Архитектура и поведение

- Рендерит статическую кнопку с фиксированным текстом "Поделиться".
- Имеет атрибут `type="button"` для предотвращения отправки форм при
  использовании внутри тегов `<form>`.
- Не управляет состоянием и не содержит обработчиков событий.

## Пример использования

```jsx
import ShareButton from '@/shared/ui/ShareButton';

<ShareButton />
```
